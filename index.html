1. Custom Bet Size Line

- Custom Bet Size: list of value separate by "," coma. each value is a bet size.
- X: is the multiplicator to reach the right amount for your bets list.
- LastBetStop: if you want to stop the bot betting when it will lost the last bet in the list then check the StopLastBet checkbox. if Multi is enable then it will stop if loosing the LastCustomBet(in thelist) X Multi X LastMulti(in thelist).

2. % Win Odd/FirstBet/Swap

- % Win/Odd: Possible value 5% to 95%.
- FirstBet: is Low or High
- Swap: Choose how the bot should swap between Lor or High. Same: wont swap

3.Repeat Losse/Win/AW/AW%

- Repeat a bet if Loss
- Repeat a bet if Win
It will repeat as many time as set by user. Default value is 1=doesnt repeat. Must be an integer over 1.
- AW: AutoWithdraw - if true Autowithdraw is enable.
- AW%: set % amount you want to autowithdraw/reinvest reaching the right profit set in General Settings. you should set your addy there too for each crypto. Cannot be negative.

4.Multiplicator list/Multi/T2C

- Multiplicator List: Will be use to multiply a loosing system by value in the list separate by ",".
- Multi checkbox: to enable the multiplicator list.
- L2C%: Loss to Cover - to set the % of your loss that you want to cover when your system get a break. if loss are covered then bot will reset to base bet and base multi.

5. High/Low Patern

Enable it with Swap=patern
Another list to fix your High/Low bets the list should have the same number of value as the Custom Bet Size list. and you have to set it with 3 differents value 0=Low, 1=High, and other values=random. As example you should set it like that 1,0,1,0,4,0,1...=High, Low, High, Low, Random, Low, High...

6.%Win Patern and Crypto choice

% Win Patern works as High/Low Patern. Enable it with swap=Patern. As to be same number value in the list as Custom Bet Size list.

Auto Bet Help

Infos: All amount should be set in Satoshi format 1000 satoshi for 0.00001 btc. All % value should be type like that 100 for 100%

Look at Example1 and Example2 button, and details, at the end of this help section...

1.CustomBetSize -X- LastBetStop

- Custom Bet Size: list of value separate by "," coma. each value is a bet size.
- X: is the multiplicator to reach the right amount for your bets list.
- LastBetStop: if you want to stop the bot betting when it will lost the last bet in the list then check the StopLastBet checkbox. if Multi is enable then it will stop if loosing the LastCustomBet(in thelist) X Multi X LastMulti(in thelist).

2. % Win Odd - FirstBet - Swap

- % Win/Odd: Possible value 5% to 95%.
- FirstBet: is Low or High
- Swap: Choose how the bot should swap between Lor or High. Same: wont swap

3. R2BBAfter - AW/AW%

R2BB=Return To Base Bet After; L=Loss ; W=Win
Check to enable R2BB - Uncheck to set increase % same as 999dice
- ResetOnWin: IncreaseOnWinPercent (optional): If ResetOnWin is false, after winning a bet, increase the next bet by %. Cannot be negative.
- ResetOnLoss: IncreaseOnLosePercent (optional) If ResetOnLose is false, after losing a bet, increase the next bet by choosen %. Cannot be negative.
- AW: AutoWithdraw - if true Autowithdraw is enable.
- AW%: set % amount you want to autowithdraw/reinvest reaching the right profit set in General Settings. you should set your addy there too for each crypto. Cannot be negative.

4. MaxBetSize - Stop - R2BB

- MaxBetSize: (optional) The maximum bet amount in satoshi format, or 0 for no maximum.
- Stop: (optional) If true, then after a losing bet where the bet amount is equal to MaxPayIn, betting will end.
- Reset: (optional) If true, then after a losing bet where the bet amount is equal to MaxPayIn, the next bet amount will be BasePayIn.

5. Max/Min Balance Stop

- MaxBal Stop: (optional) After a bet, if your balance is at least this amount, then stop betting. 0 for no maximum.
The amount set will be add to balance dynamically before a new bet.
- MinBal Stop: (optional) After a bet, if your balance is less than this amount, then stop betting.
The amount set will be add to balance dynamically before a new bet.

6. Multiplicator list/Multi/T2C

- Multiplicator List: Will be use to multiply a loosing system by value in the list separate by ",".
- Multi checkbox: to enable the multiplicator list.
- L2C%: Loss to Cover - to set the % of your loss that you want to cover when your system get a break. if loss are covered then bot will reset to base bet and base multi.

7. Replay - NumBet - Crypto

- Replay if Profit=0: After a bet if true, will repeat the same bet if bet profit=0.
- Number of Bets: The maximum number of bets to make. Current limit is 200.
- Crypto to Play: Choose the crypto you want to gamble.

Now if you are ready and you thing everything is set correctly then hit the "InjectSettings" button to inject your options for the bot.
